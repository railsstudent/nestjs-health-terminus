## Description

Create a NestJS application to set up Bull queues to process backend jobs

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Verify api in docker

```
$ docker exec -it api /bin/sh
```

## Verify redis in docker

```
$ docker exec -it redis /bin/sh

In docker container, type redis-cli and verify output is
127.0.0.1:6379>
```
